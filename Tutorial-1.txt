#persiapan
- pertama download programnya: https://github.com/yt-dlp/yt-dlp/releases/tag/2024.05.27
- download yt-dlp.exe (untuk windows)
- siapkan folder untuk program utamanya
- ketik edit the system environment pada windows taskbar. lalu klik environtment variables dan copy lokasi file yt-dlp tadi ke bagian user variables dan system variables (yang ada tulisan Path)
  Contoh: C:\Users\Syahril\Downloads\Youtube-dlp
- untuk test agar bisa atau tidak ketik yt-dlp pada command prompt. SELESAI!

cara gunakan yt-dlp (github) untuk download video atau audio youtube (dan yang lain) dengan format apapun yang tersedia menggunakan command-line

1. untuk cek format apa saja yang ada di video ataupun audio tersebut:
yt-dlp --list-formats url

2. untuk download videonya saja (perhatikan nomer id)
yt-dlp -f nomer_id url

contoh:
yt-dlp -f 391 https://www.youtube.com/watch?v=Mw1MWfilz_k&pp=ygUMdmlkZW8gNjAgZnBz

3. untuk download video+audio
yt-dlp -f nomer_id+_nomer_audio url

contoh:
yt-dlp -f 391+140 https://www.youtube.com/watch?v=Mw1MWfilz_k&pp=ygUMdmlkZW8gNjAgZnBz
